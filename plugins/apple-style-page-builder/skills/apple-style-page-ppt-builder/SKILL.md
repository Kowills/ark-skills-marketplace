# Apple Liquid Glass 设计风格的网页 PPT 生成器

## 何时使用
当用户需要生成以下内容时激活此技能：
- 符合Apple Liquid Glass语言的网页版PPT
- iOS 26风格的UI界面设计
- 网页/应用的视觉设计规范
- 圆角柔和的图表或架构图
- 苹果风格的营销/宣传材料

## 核心设计规范（必须严格遵守）
参考 iOS 26 / Apple Liquid Glass 设计语言：
Liquid Glass 材质：所有卡片、容器使用半透明磨砂玻璃效果（frosted glass），有微妙的背景模糊和折射感
大圆角：所有卡片、按钮、容器使用 20-28px 圆角
轻盈配色：浅色背景为主（接近白色但带微妙的蓝灰色调 #F5F7FA），深色模式不需要
强调色：使用柔和的蓝紫渐变（#007AFF → #5856D6）作为主强调色，辅助色用浅绿 #34C759、浅橙 #FF9500
字体：标题用 SF Pro Display Bold / 思源黑体 Bold，正文用 SF Pro Text Regular / 思源黑体 Regular
间距：大量留白，元素之间间距宽松，不要拥挤
阴影：使用极浅的投影（0 4px 20px rgba(0,0,0,0.06)），不要硬阴影
图标：使用 SF Symbols 风格的线性图标，线条粗细 1.5px
图表 / 架构图：用圆角矩形 + 浅色填充 + 细线连接，不要尖锐的箭头，用柔和的弧线箭头
背景：可以在页面底部加一个大的模糊渐变光斑（浅蓝或浅紫），增加层次感
PPT 比例：16:9
语言：中文

## 执行步骤
1. 接收用户设计需求（UI、PPT、图表等）
2. 对照上述规范逐项应用设计要求
3. 生成符合规范的设计方案/代码/布局
4. 提供设计决策说明（如为何选择24px圆角而非28px）
5. 输出完整设计文件/代码/描述
